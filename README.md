# Arena-Cash-Cursor

Protótipo navegável do MVP de onboarding da ArenaCash.

## Como abrir localmente

```bash
python3 -m http.server 5174
```

Depois acesse:

```text
http://localhost:5174
```

## Escopo

- Hub de onboarding com checklist e progresso.
- Etapa de perfil com explicação contextual.
- Estado de verificação pendente.
- Escolha da primeira aplicação.
- Tela de sucesso da primeira aplicação.

## Deploy

Este projeto é um site estático e pode ser publicado pelo GitHub Pages a partir da branch `main`.
